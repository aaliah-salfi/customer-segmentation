# Customer Segmentation using K-Means Clustering

## 📌 Project Description
This project uses K-Means Clustering algorithm to segment customers based on their Annual Income and Spending Score.

## 📊 Objective
To divide customers into different groups for better marketing strategies.

## 🧠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## 📁 Files Included
- customer_segments.csv
- Jupyter Notebook (.ipynb)

## 🚀 Outcome
Customers are divided into 5 clusters:
- Luxury Customers
- Careful Spenders
- Impulsive Buyers
- Budget Customers
- Average Customers
